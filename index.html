Smart Connector

Created by Ernesto Celis / @celisdelafuente

Features

  • Smart Connector enables you to extract data from assorted sources
  • Transforms data into an HL7 messages to be delivered to another systems
  • HTML5 Drag & Drop Web User Interface
  • RESTful API
  • Runs standalone, within an existing application server or can be loaded as dependency for any Java or Clojure project

Adapters

  • PostgreSQL, Oracle, MySQL, SQL Server Database Adapters
  • Mirth Connect Adapter (WIP)
  • Cirrus Adapter (WIP)

WIP == Work In Progress

Data Storage

  • Plain text files in YAML for connection schemas
  • Logstash for event logs (WIP)
